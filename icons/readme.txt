　概要
────────────────────
　UWSC各種アイコンをモダンな感じに再構成したロゴセットです

　関連付けアイコンは他のファイルと並んだときに不自然にならないように
　Windows のファイルアイコンを参考にしました

　一部アイコンは大きいサイズをWindowsにスケーリングさせたほうがきれいに表示されるかも

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
　内容
────────────────────
　readme.txt
　　このファイル
　UWSCR_FHD_*.png
　　アイコン等に使いやすいように数パターンあります
--------------------
　UWSC\
　　uwsc.exe 関連
　　uwsc.exe自身 と 関連付けアイコン
　　白文字 と 黒文字
　EXEUWS\
　　exeuws.exe 関連
　　uws2exe で exe化したデフォルトアイコンがこれになってました
　　exeuws.exe自身 と それに模したファイル型(関連付けアイコンの色違い)
　　黒文字 と 文字なし
　UWS2EXE\
　　uws2exe.exe 関連
　　uws2exe.exe自身 と それに模したファイル型(関連付けアイコンの色違い)
　UDEBUG\
　　udebug.exe 関連
　XREF\
　　xref.exe 関連
　UI\
　　UI 関連
　　uwsc.exe, udebug.exe 等のUIアイコン
　_ScriptLab Shadow Setting\
　　影をつける際のマクロ
　　使用するには pyrochild Plugin Pack から以下が必須
　　　ScriptLab
　　　Outline Object
--------------------
　pdn\
　　Paint.NET Image ファイル
　　PhotoshopでいうPSD
　　レイヤー情報とか保持してます
　png\
　　pdnをpngに書き出す際に影を付けたもの
　ico\
　　pngをマルチアイコンにしたもの
