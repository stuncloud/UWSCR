数学関数#

isnan(数値)#
値が NaN であるかどうかを調べる
パラメータ:

数値 (数値) -- 調べる値

戻り値:

NaN であればTRUE

サンプルコード

print IsNan(NaN)  // True
n = NaN
print IsNan(n)    // True
print IsNan(1)    // False
print IsNan("あ") // False

注釈

戻り値の型:

数値

戻り値:

得られたランダム値
